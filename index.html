Selected Project

Artifact 6 — Machine Learning Algorithms Visual Framework

Introduction: Machine Learning is an important branch of Artificial Intelligence that allows computers to learn patterns from data and make predictions without being explicitly programmed for every task. This project focuses on understanding several common machine learning algorithms and organizing them into supervised and unsupervised learning categories.

Description: This artifact is an infographic presenting ten machine learning algorithms, divided into supervised (Decision Tree, Random Forest, SVM, KNN, Neural Networks) and unsupervised (K-Means, DBSCAN, PCA, Apriori, Hierarchical Clustering). Each algorithm includes applications and real-world examples.

Objective: Provide a visual framework to help viewers understand algorithm categorization and applications.

Process: I researched commonly used algorithms, organized them into groups, and designed the infographic for clarity.

Tools: ChatGPT, Online Research, Infographic Design Tools

Value Proposition: Helps students, educators, and professionals understand how algorithms work and when to use them.
